Audio

For audio, we have mp3, m4a, ogg.

  • type=mpeg: MP3
  • type=x-aac: M4A
  • type=ogg: OGG

HTML Code

<!--MP3-->
<audio controls='controls'>
  <source src='/media/audio.mp3' type='audio/mpeg'>
</audio><br>

<!--M4A-->
<audio controls='controls'>
  <source src='/media/audio.m4a' type='audio/x-aac'>
</audio><br>

<!--OGG-->
<audio controls='controls'>
  <source src='/media/audio.ogg' type='audio/ogg'>
</audio>

Result



Not all web browsers support all audio types, so, to overcome this, we need to define all three, but don't worry, because web browser will pick only one.

HTML Code

<audio controls='controls'>
  <source src='/media/audio.mp3' type='audio/mpeg'>
  <source src='/media/audio.m4a' type='audio/x-aac'>
  <source src='/media/audio.ogx' type='audio/ogg'>
</audio>

Result

Video

For video, we have mp4, webm, ogv.

  • type=mp4: MP4
  • type=webm: WEBM
  • type=ogg: OGV

HTML Code

<video controls='controls'>
  <source src='video.mp4' type='video/mp4'>
</video><br>

<video controls='controls'>
  <source src='video.webm' type='video/webm'>
</video><br>

<video controls='controls'>
  <source src='video.ogv' type='video/ogg'>
</video>

Result



Not all browsers support all those types, so, we can arrange them together, then browser will pick one.

HTML Code

<video controls='controls'>
  <source src='/media/video.mp4' type='video/mp4'><br>
  <source src='/media/video.webm' type='video/webm'><br>
  <source src='/media/video.ogx' type='video/ogg'>
</video>

Result

autoplay

This tag initiates the video or audio when the page renderizes.

HTML Code

<audio controls='controls' autoplay>
  <source src='/media/audio.mp3' type='audio/mpeg'>
</audio>

Result

loop

This attribute permits audio or video playing all the time.

HTML Code

<video controls='controls' autoplay loop>
  <source src='/media/video.mp4' type='video/mp4'>
</video>

Result

muted

This attribute permit audio or video initiates muted.

HTML Code

<audio controls='controls' muted>
  <source src='/media/audio.mp3' type='audio/mpeg'>
</audio>

Result

preload

Permit to load each by each the video or audio.

<video controls='controls' preload>
  <source src='/media/video.mp4' type='video/mp4'>
</video>
