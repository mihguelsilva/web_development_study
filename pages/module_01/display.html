display: block

  • Ocupy father's 100% height;
  • It's positioned bellow before's element;
  • It's possible to use width and height;
  • It can define margin-top-right-bottom-left.

Tags examples:

  • <p>
  • <h1>
  • <ul>
  • <li>
  • <div>
  • <article>
  • <section>

display: inline

  • Ocupy height of it's before's element;
  • Inline's elements are positioned at the same line;
  • It's not possible use width and height;
  • If applied float's attribute, inline's elements are transformed to block;
  • It could define online margin-right-left.

Tags examples:

  • <a>
  • <span>
  • <i>
  • <em>

display: inline-block

Elements are both inline and block.
