<input autocomplete>

When data are sent from forms, those data are saved from browsers. Sometimes, some data are used so many times, that when we access an different input, where input's name are email, automatically, appears, bellow that input, what you are tipping. When this happens, it's because that input has autocomplete's attribute set as one (on). To take off this attribute, you must set autocomplete as off.

HTML Code

<form>
  Name: <input type='text' placeholder='Name complete' name='name' autocomplete='on'><br>
  Address: <input type='text' placeholder='street,number,neighborhood' name='address'><br>
  City: <input type='text' name='city'><br>
  Phone: <input type='text' name='phone' autofocus autocomplete='off'>
</form>

Result
