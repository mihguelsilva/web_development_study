<table></table>

To create an table, we must declare a table tag.

<table>
</table>

<tr></tr>

Tables has lines and columns. To create a line, we must declare <tr></tr> tag.

<table>
  <tr></tr>
</table>

<td></td>

To create a column, we must declare <td></td> tag.

<table>
  <tr>
    <td></td>
  </tr>
</table>

<caption></caption>

We can insert a title into an table. To do this, we must use <caption></caption> tag.

HTML Code

<table>
<caption>Table's title</caption>
  <tr>
    <td>first line, first column</td>
    <td>first line, second column</td>
  </tr>
</table>

Result

Table's title
first line, first column first line, second column

boder-collapse: separate

Separate it's the default for border-collapse table's attribute. This attribute maintain table's lines and columns separated.

HTML Code

<table>
<caption>Table's title</caption>
  <tr>
    <td>first line, first column</td>
    <td>first line, second column</td>
  </tr>
  <tr>
    <td>second line, first column</td>
    <td>second line, second column</td>
  </tr>
</table>

CSS Code

table {
  border-collapse: separate;
  border: 1px solid black;
}
table td {
  border: 1px solid black;
}

Result

Table's title
first line, first column first line, second column
second line, first column second line, second column

vertical-align

This attribute align in vertical table's texts. We can define three diferent values:

  • middle: default value;
  • top: set text at top;
  • bottom: set text at bottom

  • HTML Code

    <table>
    <caption>Table's title</caption>
      <tr>
        <td rowspan='2' id='vertical'>first line, first column</td>
        <td>first line, second column</td>
      </tr>
      <tr>
        <td>first line, second column</td>
      </tr>
    </table>

    CSS Code

    table {
      border-collapse: collapse;
      border: 1px solid black;
    }
    table td {
      border: 1px solid black;
    }
    table td#vertical {
      vertical-align:top;
    }

    Result

    Table's title
    first line, first column first line, second column
    second line, second column
